How to Assign Audio Modes

You must assign an audio mode to each audio device that you plan to use with Cisco IP Communicator:

  • Headset mode
  • Speakerphone mode
  • Handset mode
  • Ringer mode

Audio mode selection tells Cisco IP Communicator which audio devices you want to use for audio input and output.

The first time that you launch Cisco IP Communicator, you can assign audio devices to audio modes by using the Audio Tuning Wizard. Subsequently, you can assign audio devices to modes by right-clicking Cisco IP Communicator and choosing Preferences > Audio tab.
