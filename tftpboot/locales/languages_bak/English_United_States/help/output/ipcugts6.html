Capturing Information About Problems

If Cisco IP Communicator should unexpectedly close, the Problem Reporting Tool automatically launches and captures relevant data for troubleshooting purposes. Use this procedure to send the report to your system administrator.

  1. Follow the instructions in the Problem Reporting Tool to describe the problem. Make sure to include this information:
    • A description of the problem.
    • An explanation of what you were doing at the time the problem occurred.
    • Which audio device was in use at the time.
    • Any other details that might have affected the situation.
  2. Look on your desktop for a file named CIPC-ProblemReportxxx.zip, where xxx is a number.

If you experience other problems and the problem report is not automatically generated, your system administrator might request log files. Unlike the QRT (which reports what the problem is), these logs provide detailed information that help troubleshoot the problem. Use this procedure to collect these files:

  1. Enable detailed logging (right-click > Preferences > User tab, and check Enable Logging).
  2. Attempt to reproduce the problem. If you are unable to reproduce the problem, the logs will not have detailed information.
  3. Create the report by choosing Start > All Programs > Cisco IP Communicator > Create CIPC Problem Report.
  4. Follow the instructions you see to describe the problem. Make sure to include this information:
    • A description of the problem.
    • An explanation of what you were doing at the time the problem occurred.
    • Which audio device was in use at the time.
    • Any other details that might have affected the situation.
  5. Before you click Finish, note the name of the file that has been created on your desktop.
  6. E-mail this file to your system administrator.

    7. Tip     Through the Audio Tuning Wizard, you can launch the Problem Reporting Tool to report audio issues. Click the top left corner of the Audio Tuning Wizard title bar, and choose Troubleshooting Info. A pop-up message asks you if you want to launch the Problem Reporting Tool.
