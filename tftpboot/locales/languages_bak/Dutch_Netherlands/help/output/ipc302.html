Basic Call Handling

This section covers basic call-handling tasks such as placing, answering, and transferring calls. The features required to perform these tasks are standard and available on most phone systems.

This section contains the following topics:

  • Placing a Call

  • Answering a Call

  • Ending a Call

  • Using Hold and Resume

  • Using Mute

  • Transferring a Connected Call

  • Switching Between Calls

  • Forwarding Your Calls to Another Number

    Tip     For more information about placing, receiving, and ending calls while using a handset, speakerphone, or headset, see the "Using Headsets and Other Audio Devices" topic.
