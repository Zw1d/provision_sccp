Installing Cisco IP Communicator on Your Computer

Your system administrator will provide you with an executable file or an installation link. Follow the procedure below to install the Cisco IP Communicator software on your computer.

Note     If you use a laptop computer, be sure that you are not connected to a docking station when launching Cisco IP Communicator for the first time after installation.

Procedure

  1. Open CiscoIPCommunicatorSetup.exe by double-clicking on this file or on the installation link provided by your system administrator.

    The InstallShield wizard opens and begins preparing for installation.

  2. Choose the language in which you would like to use Cisco IP Communicator from the drop-down list.

  3. When prompted, click Next to initiate the InstallShield wizard.

  4. Read the license agreement carefully, then click "I accept" and Next.

  5. Enter data in the Customer Information window and click Next. If you think other people need to use Cisco IP Communicator on this computer, select yes.

  6. In the Ready to Install window, click Install.

    Installation might take a few minutes.

  7. If you want to launch Cisco IP Communicator now, click the "Launch the program" check box and click Finish. (In some cases, you will be prompted to reboot at this point and will not see the "Launch the program" check box.)

See the "Launching Cisco IP Communicator" topic.
