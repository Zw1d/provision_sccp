Advanced Call Handling

Advanced call-handling tasks involve special (non-standard) features that your system administrator might configure for you to use on Cisco IP Communicator, depending on your call-handling needs and work environment. You will not have access to these features by default.

This section contains the following topics:

  • Redirecting a Ringing Call to Cisco IP Communicator

  • Making Conference Calls

  • Understanding Shared Lines

  • Adding Yourself to a Shared-Line Call
