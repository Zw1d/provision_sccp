Note     You should assign a USB handset to handset mode; this configuration allows Cisco IP Communicator to recognize if the handset is on-hook or off- hook, allowing you to end a call by hanging up the USB handset, for example. For more information about this assignment, see the "Understanding Audio Modes" topic.
