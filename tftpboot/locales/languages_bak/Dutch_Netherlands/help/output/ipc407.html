Viewing and Customizing Preferences

Most Cisco IP Communicator settings can be accessed from windows in the Preferences dialog box. To access Preferences, enter the Alt + S keyboard shortcut, click the Menu button, or right-click on Cisco IP Communicator and choose Preferences.

The topics below describe the windows in the Preferences dialog box:

  • User Settings

  • Network Settings

  • Audio Settings

  • Directories Settings
