Hardware and Platform Requirements

To use Cisco IP Communicator, you need a computer with a sound card and/or USB audio device. The tables that follow shows the minimum and recommended requirements for the computer and operating system on which Cisco IP Communicator runs. Providing a faster CPU and additional RAM (recommended configuration) allows Cisco IP Communicator to be more responsive and to launch faster

In addition to requirements outlined in the table above, Cisco IP Communicator requires the following:

  • A minimum of 100 MB free disk space

  • A non-ISA full-duplex sound card or USB audio headset or handset

  • A 10/100 Mbit Ethernet network interface card
