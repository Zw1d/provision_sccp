Optimaliseren voor geringe bandbreedte

Wanneer u Cisco IP Communicator gebruikt via een verbinding op afstand, bijvoorbeeld via een VPN-verbinding vanaf huis of vanuit een hotel, wordt de stemkwaliteit waarschijnlijk nadelig beïnvloed door de geringe bandbreedte. Wanneer u Cisco IP Communicator gebruikt via een verbinding op afstand, kunt u voorkomen dat de audio metalig klinkt door de functie Optimaliseren voor geringe bandbreedte in te schakelen.
