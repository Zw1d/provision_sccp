An Overview of Settings

Here is some useful information to keep in mind about Cisco IP Communicator settings:

  • Most settings are accessible by choosing Preferences from the menu. You can access the menu from the menu icon in the window control button bar, or by right-clicking anywhere on the interface. See the "Viewing and Customizing Preferences" topic.

  • Ring sounds and background image settings are available from > User Preferences. See the "Customizing Rings and Message Indicators" topic.

  • Most settings are accessible on your IP Communicator, but a few are accessed online from your User Options web pages. See the "Logging In to the User Options Web Pages" topic for more information.

  • If is not responsive, your system administrator might have disabled this button on your phone. Ask your system administrator for more information.
